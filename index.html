HTML


IDEAS GENERALES

HTML dentro de la dupla que representa para las paginas web es el texto presente en ellas codificado mediante etiquetas básicas como lo son

  • h1-6>
  • p>
  • ul>, ol>, li>

Las cuales representan los títulos y subtítulos, párrafos y listas respectivamente.

APUNTES

HTML dice el profe es la piel de un monito de lego, yo diría que es la pluma para una hoja cotidiana, simple y claro.



RESUMEN

HTML es lo que indica el ordenado de las páginas, es la estructura de esta contando con los textos, imágenes y su contenido.


CSS


IDEAS GENERALES

Cascading Style Sheets o CSS es la separacion del contenido y el estilo en donde se puede modificar la apariencia de una pagina web html

APUNTES

CSS es el esqueleto del monito de lego, las plumas de colores para la cartulina diría yo; en CSS modificamos el apartado visual de la pagina web, donde se ubican exactamente (con pixeles) cada cosa, su color y si cuenta con alguna interacción o función extra leve. Es un código en cascada puesto que funciona o se escribe de forma vertical. Es el editor del HTML


RESUMEN

En CSS enchula el HTML, le da mejor aspecto visual.

ETIQUETA IMG


IDEAS GENERALES

Sirve para insertar imágenes (src) en una página, también debe de contar con una descripción clara (alt) y si es necesario una fuente de donde se obtuvo la imagen. Más allá del código las imágenes deben estar guardas en una carpeta aparte dentro de la carpeta del html.

APUNTES

Las imágenes deben estar guardadas con un nombre corto y fácil de ubicar, deben de tenerse guardadas en el menor tamaño posible lo cual podemos conseguir de forma rápida en Photoshop.


RESUMEN

Mediante la etiqueta img conseguimos insertar una imagen en la web.


TEXTO


IDEAS GENERALES

Son las etiquetas que se le añaden a los textos las cuales sirven para resaltar de cierta forma el texto seleccionado.

  • b> texto en negritas
  • strong> texto en negritas
  • i> texto en italica
  • em> texto en enfasis
  • mark> texto marcado
  • del> texto tachado

APUNTES

Sirven para los textos de los párrafos resaltar ciertas palabras, el profe hablo de un ejemplo sobre el cambio de fecha de un concierto para una pagina en donde rayaba el texto previo para aclarar que la fecha inicial ya no contaba.
